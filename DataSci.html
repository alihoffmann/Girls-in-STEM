What is a Data Scientist?

Data Scientist (n): a person who has the knowledge and skills to conduct sophisticated and systematic analyses of data. A data scientist extracts insights from data sets for product development, and evaluates and identifies strategic opportunities. src: bigdata-madesimple.com




But what does this mean?

Data scientists are individuals who analyze data. Data scientists also perfrom data mining and retrieval processes on a large amount of data to determine trends and other relevant information. The trends data scientists use are important for companies to use to maximize their profits. This data is stored in data centers and solve business problems, optimize performance, and enable companies to gather intelligence. src: technopedia.com

Let's break that down a bit further...

Data scientists analayze big data. Big data is the name given to large data sets that can be analyzed using computers to discover trends. These trends can be used to relate human behavior and interactions. Essentially, the goal of data scientists' is to analyze large sets of data to find trends.

Once data scientists find these trends, data scientists return to a set of data to try and find more trends. Oten, data scientists are given data and asked to analyze it in hopes of finding trends. Once these trends are found, they can be used to maximize profits and reach a large amount of people. This helps companies to expand and to make more money.
