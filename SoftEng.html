What is Software Engineering?

Software Engineering (n): the process of analyzing user needs and designing, constructing, and testing end user applications that will satisfy these needs through the use of software programming languages. It is the application of engineering principles to software development. src: techopedia.com




But what does this mean?

Software engineers work on large, complex systems which businesses and organizations need in order to function. Their job is much more complicated than creating a simple program. Instead, they create all portions of a program and are constantly working to modify it. Software engineers work in teams to determine which software to create, how to create it, and to create it. In larger companies, software engineers often work with user interface designers to create products which are appealing to people.

Let's break that down a bit further...

Software engineers focus on benefitting the user. They understand what the user needs to maximize their use of the software and the engineers uses this knowledge to modify the software to meet consumers' needs.

Software engineering is slowly becoming more common because the hardware needed to run the software is becoming less expensive and easier to make. Consequently, it is common for large software systems to be more complex than the hardware used to host the software. This has created a large demand for software engineers. Software engineers' products have the ability to change a user's experience and can harm a software's effectiveness if the product is poorly created or if consumers do not like the product.
